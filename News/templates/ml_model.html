Details

News plays an essential role in our daily life. The goal of this project is to develop an interactive, web-based visual application that delivers top headlines news by sentiment and category, where sentiment is informed by a sentiment detection machine learning model, from news sources around the United States.

The idea behind a sentiment-driven news app is to provide the user, the choice to explore and control the news they consume, classified further by sentiment.

Sources:
  • Training dataset - https://www.kaggle.com/rmisra/news-category-dataset
  • Testing dataset - https://newsapi.org/

The execution of the project is comprised of the following stages:

  • Data Exploration - Collect the latest top headline news articles from around various news sources in United States using News API.
  • Pre Processing - Make use of various Natural Language Processing techniques such as removing non-alphabetic characters and stop words (a set of very common words like the, a, and, etc.) in the article text and vectorizer to transform word representations to numerical versions that a model can interpret. Generate a summary of the processed article text using a summarizer.
  • Model Evaluation and Validation - Employ various Machine Learning models against the training data set, choose a model yielding the best accuracy and use it on the summary generated above to predict the sentiment.
  • Create a web application using Python, Flask, a centralized cloud database with AWS for hosting the data, and deploy it on Heroku.
Of all the several different machine learning models and classifiers that we tested with, the Support Vector Machine (SVM) model performed the best and yielded higher accuracy scores.
